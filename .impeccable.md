## Design Context

### Users

People who buy punk-leftist die-cut vinyl stickers and want them on laptops, lockers, gas pumps, hearses, and the back of every cop car they can find. Politically left, anti-cop, queer-supportive, anti-Sticker-Mule (CEO is MAGA-aligned). They aren't shopping a Shopify store for the mall version — they're following a one-person operation that prints stickers in a Denver basement, ships them by hand, and donates half the profits to bail funds. Trust + authenticity matter more than slickness.

Use context: phone or laptop, browsing casually, often after seeing a link on social. Buying decision is fast — sticker shopping isn't a research-heavy purchase. Cart is small (single-digit items, $4 each, free shipping over $40). They came for one sticker; the site's job is to delight them into buying three more.

### Use cases

- **Browse + buy:** scan the catalog, pick stickers, check out. Stripe hosted Checkout when wired (phase 5).
- **Read the manifesto:** confirm this isn't a corporate operation; absorb the politics.
- **Join the wake (newsletter):** stay in the loop on restocks + new drops.
- **(future) flea-market dates, mutual-aid log, press inquiries** — not in scope yet.

### Brand personality

Three words: **punk · leftist · positively-damned.**

Voice: blunt, not edgy-for-edgy's-sake. Funny without being precious. Self-aware that it's a one-person sticker shop, not a movement. Will say "Hail thyself" but also says "Cart" instead of "Coffin" because thematic substitutions for utility UI are lazy.

Emotional goals: belonging (you're with the freaks), permission (yes you can slap this on the wall at work), small joy (a 3" plastic act of refusal). NOT urgency, NOT scarcity, NOT FOMO.

### Aesthetic direction

**Horror-Punk Pop-Art.** Locked. Channels:

- Misfits gig flyers (Crimson Ghost, late-70s / early-80s)
- Vampira + Elvira blacklight horror-host portraits
- Misfits "3 Hits from Hell" tape-strip eye-cutout album cover
- Moon-of-Jupiter-style neon-on-black acrylic / spray paint portraits
- Siouxsie Sioux stark red-on-black portraiture

Theme: **dark.** Pure pitch black `#000000` background, bone-cream body text, rotating acid neon accents per section / card (hot pink, electric blue, acid lime, acid yellow + reserved warning red). Hand-painted brand wordmark in Permanent Marker. Bowlby One chunky display for h1/h2/h3 and prices. Special Elite typewriter for zine / photocopy moments. Inter for body.

Geometry: hard offset blocky shadows, heavy 2-3px ink-line borders, hard-cornered cards (`--radius-tight: 2px`), pill reserved for nav cart only, surgical rotations on taped eyebrows + photocopy labels + card brand strips.

Anti-references:

- Cinzel-style monumental engraved serifs (was the prior "goth boutique" direction — dead)
- Soft drop shadows with blur
- 14/18px or pill-rounded panels (except the cart pill)
- Hairline borders on dark surfaces
- Atmospheric body gradients or grain overlays
- Ember-orange anywhere (old direction; tokens are gone)
- Acid-red used decoratively (reserved for warning semantics: sold-out, sale, removal)
- Mixing multiple acid accents inside one element (multiple can coexist in a frame, never blended in one element)
- "Punk-moment" patterns (tape, stencil, ransom-note collage) scattered as ambient texture — they live in defined slots only

### Accessibility

- All interactive elements get `:focus-visible` with `outline: 3px solid var(--color-acid-blue); outline-offset: 3px;` (the new universal pattern from Tasks 2-9).
- Text contrast: bone-cream (`#f4ecd8`) on pitch black is ~17:1. Acid-on-ink combinations all pass WCAG AA+ (pink/ink ~7.5:1, lime/ink ~13.5:1, yellow/ink ~17:1).
- Heading landmarks: when a section drops its `<h2>` per spec (e.g., Manifesto), add `aria-label` to the `<section>` to preserve screen-reader navigation.
- Mobile: nav collapses to "cart only" below 640px (hamburger toggle is out of scope until v2).
- Reduced motion: not currently handled — interactions are 1-2px translates, sub-vestibular. Add `@media (prefers-reduced-motion: reduce)` overrides when motion is added beyond what's already shipped.

### Design Principles

1. **Punk in defined slots, not as ambient texture.** Tape, stencil, ransom-note, splatter — each has a specific home (eyebrows, card brand strips, hero, etc.). Repeating them everywhere kills the meaning.
2. **One acid per element, multiple in a frame.** A card has one strip color. A section has one eyebrow color. The page as a whole rotates through 4-5 acids. Never blend two acids inside one element.
3. **Hard edges, hard shadows, no soft polish.** Pop-art posters are flat-color print, not cinematic. No blur, no drop-shadow filters, no peel-back animations. Block shadows offset by 4px in flat color, no rgba.
4. **Bone-cream is the only text on dark surfaces. Ink-black is the only text on acid surfaces.** Never mix.
5. **Utility UI uses plain words.** Cart says "Cart," not "Coffin." Checkout says "Checkout," not "Last Rites." Voice lives in headlines, eyebrows, manifesto, product names, error states — wherever it earns its place.
6. **Refuse the AI defaults.** No Inter for display. No Fraunces/Cormorant/Playfair. No purple-to-blue gradients. No gradient text. No `border-left: 4px solid X` accent stripes. No nested cards. If a font/pattern/color would look "designy," interrogate it.

## Token reference (quick)

```css
/* Base */
--color-pitch: #000000; /* page bg */
--color-ink: #050505; /* text on acid surfaces */
--color-bone: #f4ecd8; /* text on dark, ink-line outlines */

/* Acids — rotating */
--color-acid-pink: #ff2d8a;
--color-acid-blue: #00d4ff;
--color-acid-lime: #a3e635;
--color-acid-yellow: #fff200;
--color-acid-red: #e3151f; /* WARNING SEMANTICS ONLY */

/* Type */
--font-brand:
  "Permanent Marker", ...; /* hand-painted brand + one signature word per section */
--font-display: "Bowlby One", ...; /* h1/h2/h3, prices */
--font-zine: "Special Elite", ...; /* eyebrows, micro-CTAs, footer caps */
--font-body: "Inter", ...; /* body, nav, card titles */

/* Geometry */
--radius-tight: 2px;
--radius-pill: 9999px; /* cart only */
--shadow-block-bone: 4px 4px 0 var(--color-bone);
--shadow-block-ink: 4px 4px 0 var(--color-ink);
--shadow-block-pink: 4px 4px 0 var(--color-acid-pink);
--shadow-block-blue: 4px 4px 0 var(--color-acid-blue);
--border-bone: 3px solid var(--color-bone);
--border-ink: 2px solid var(--color-ink);
--rotate-tape: 6deg;
--rotate-stencil: -2deg;
--rotate-strip: -1deg;
```
